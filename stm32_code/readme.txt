	一个主模块可以带好多子模块
	
	
	
	大体功能：
	1个主模块，可以挂接多个子模块，为了保证通信的及时性，认为64个模块足够了，但系统可扩展为256个。
	
	
	
	
	
	
	
	通信协议的制定
	1.自动连接性， 主模块能自动找到存在它周围的子模块，并与之联接进行通信
	  1. 主机具有呼寻功能， 定期向周围发出呼寻， 有主的模块将不再返回任何信号，无主的模块随机应答。
	     主机接到应答以后， 返回确认信息，模块收到确认后，则认为是连接成功。主机在发送呼寻时，同时发出自己的信息，
		 
	  
	2.子模块中无物理地址，它能自动从连接设备中获得物理地址。
	3.主模块可以向子模块发送广播信号
	4.主模块可以独立访问每个子模块
	5.一个主模块可以最多带64个子模块
